## CLONE

```bash
git clone git@github.com:diegovera96/Practica02.git
```

## BACKEND

```bash
cd first-api-iwm
Create DB in mysql
Rename .env.example to .env and configure
composer install
php artisan key:generate
php artisan migrate
php artisan db:seed
php artisan serve
```

## FRONTEND
```bash
cd frontend/my-app
npm install axios
npm install react-bootstrap bootstrap
npm start
```
